# Решения домашних заданий по Python

В этом репозитории находятся решения заданий:

- `solution.py` — программа для подсчёта суммы цифр в строке.
- `ladder.py` — рисование лестницы из символов.
- `equation.py` — решение квадратного уравнения с коэффициентами.

Каждая программа принимает аргументы командной строки или запрашивает ввод с клавиатуры.
